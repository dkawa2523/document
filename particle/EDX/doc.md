1. 1ページ要約

発明の要旨

本技術は、半導体製造装置のウェハ上に付着したパーティクル又はコンタミについて、元素ごとの複数ピーク候補エネルギーを同一レコードに保持する簡易辞書を起点に、人工 EDX スペクトルを大量生成し、その人工データで学習した AI モデルにより、実測 EDX スペクトルから成分候補と擬似成分比を推定する技術である。さらに、モデル出力だけでなく、辞書に登録された複数アンカー近傍の一致度を用いて信頼度を算出し、低信頼度のものだけを再測定又は高忠実度解析へ回す構成とする。DTSA-II は analytical/Monte Carlo 両方の EDS スペクトルシミュレーション、Filter-fitting、XPP 補正を備え、eXSpy は EDS の背景処理・line intensity 抽出・curve fitting を備えるため、これらは本技術の拡張実施形態や比較基準として位置付けやすい。  ￼

新規性の核 3 点（推定）
	1.	簡易辞書の共通インターフェース化
元素ごとに energy_1~energy_n を同一レコードで保持し、その同一辞書を、人工スペクトル生成、AI 学習、推論時のアンカー照合、信頼度算出に共通利用する点。
	2.	人工データの作り方そのもの
単なるガウス和ではなく、線欠落、近接ピーク重なり、エネルギードリフト、背景揺らぎ、低エネルギー減衰、基板混合をパラメータ化して、半導体装置パーティクルに近い失敗モードを意図的に増やす点。
	3.	信頼度の定義
AI の出力確率だけではなく、辞書アンカー一致数、主線/副線の整合、低エネルギー危険フラグ、基板混合疑い、必要に応じた再評価ルーティングを組み合わせて信頼度を出す点。

2. 従来技術（背景）と先行技術カテゴリ

2-1. 技術背景

半導体製造装置に付着するパーティクルの EDX 成分解析では、近接ピーク、弱い副線、低エネルギー線の吸収、基板寄与、測定時間、標準試料の制約が重なり、単純なピーク検出や分布 fitting だけでは安定しない。NIST DTSA-II はその対策として analytical/Monte Carlo のスペクトルシミュレーション、Filter-fitting、XPP 補正を提供している一方、実運用ではこれを全欠陥に常時適用すると重くなりやすい。eXSpy/HyperSpy は line intensity 抽出や curve fitting を提供するが、人工学習データ生成と信頼度制御までを一体化した仕組みではない。  ￼

公開文献・公開特許カテゴリを見る限り、シンプルな元素-複数エネルギー辞書を、人工 EDX データ生成、AI 学習、推論、信頼度算出に一貫して使う構成は、少なくとも今回確認した範囲では中心的には見当たらない。また、SEM は画像単独分類や空間融合の文脈では多く見られるが、SEM を幾何 prior として低次元特徴に圧縮し、EDX 側の探索空間を狭める補助入力とする構成も、本件の差別化候補になり得る。もっとも、これは網羅検索前の推定であり、正式な先行技術調査は要確認である。  ￼

4. 提案手法

4-1. システム構成（文章）

本提案は、複雑にしすぎないことを優先し、以下の 5 機能で構成する。
	1.	辞書管理部
元素ごとに複数候補エネルギーを 1 レコードで保持する。
例: element, energy_1_keV, energy_2_keV, energy_3_keV
	2.	人工データ生成部
上記辞書のみを必須入力として、擬似 EDX スペクトルを大量生成する。初期版では line energy とピーク幅、背景、ノイズ、線欠落、ピーク重なり、基板混合を用いる。
	3.	学習部
人工スペクトルとラベルを用いて、成分有無と擬似成分比を推定する AI モデルを学習する。
	4.	推論・信頼度算出部
実測 EDX スペクトルを入力し、AI 推論結果に加え、辞書アンカー近傍の一致度、複数アンカーの整合度、低エネルギー危険度を用いて信頼度を算出する。
	5.	任意拡張部
clean wafer 又はプロセス前後の実測スペクトル、SEM 特徴量、DTSA-II/PENELOPE 等による高忠実度シミュレーションを接続する。

この構成の思想は、高忠実度物理計算を本番で常時回さず、軽い辞書と学習器を主経路に置き、低信頼度だけを詳細解析へ回す点にある。DTSA-II は高精度なスペクトルシミュレーションと定量に向く一方、eXSpy は前処理や line intensity 抽出に向くため、これらは拡張実施形態や比較実験に適する。  ￼

4-2. 関係シミュレーションでのデータ収集の仕様

初期版の前提
	•	必須入力は スペクトルエネルギー辞書のみ
	•	形状、詳細基板構造、標準試料は初期版では必須としない
	•	ただし、基板混合率は乱数パラメータとして与える

人工 EDX スペクトル生成式（初期版）
初期版の学習データ生成は、次式で表せる。

x(E)=B(E)+\sum_{e}\sum_{j} c_e \, m_{e,j}\, w_{e,j}\, G(E;E_{e,j}+\Delta E,\sigma)+\alpha S_{\text{sub}}(E)+\eta(E)

ここで、
	•	E_{e,j}: 辞書に登録された元素 e の第 j 候補エネルギー
	•	c_e: 元素寄与係数
	•	m_{e,j}: line dropout 用の 0/1 マスク
	•	w_{e,j}: 主線/副線の相対重み（初期版では簡易係数）
	•	G: ガウス又は擬似 Voigt 形状
	•	\Delta E: エネルギードリフト
	•	B(E): 背景
	•	\alpha S_{\text{sub}}(E): 任意の基板混合項
	•	\eta(E): ノイズ

初期版で必ず入れる擾乱
	•	line dropout: 同じ元素でも一部のピークを欠落させる
	•	peak overlap: 近接エネルギーの元素を意図的に混合する
	•	energy drift: エネルギー軸のずれ
	•	background variation: 背景強度のばらつき
	•	noise: Poisson または近似ノイズ
	•	substrate mixture: 粒子 + 基板の混合

保存するラベル
	•	元素 presence の multi-label
	•	擬似成分比
	•	主成分ラベル
	•	近接ピーーク危険フラグ
	•	line dropout フラグ
	•	低エネルギー危険フラグ
	•	基板混合率

将来拡張
将来拡張では、DTSA-II による analytical/Monte Carlo スペクトル、xraylib による line energy/edge/fluorescence data、SRD 164 による内殻電離断面積、XCOM による減弱係数を使って、上式の w_{e,j} や吸収項を物理化できる。DTSA-II は absolute basis でのスペクトル比較が可能で、SRD 164 は K 殻および L/M subshell の ionization cross section を提供し、XCOM は元素・化合物・混合物の attenuation coefficient を提供する。  ￼

4-3. SEM,EDX の実測データの利用の拡張

本件の拡張実施形態では、SEM/EDX 実測データを以下の 3 通りで使う。

(a) clean wafer / プロセス前後スペクトルの利用
プロセス前スペクトル又は clean wafer スペクトルを基板参照として保持し、後工程スペクトルを「基板 + 付着成分」の混合として扱う。

x_{\text{after}}(E)\approx \beta x_{\text{before}}(E+\Delta E)+\sum_k \gamma_k P_k(E)+B(E)

この使い方は、標準試料を大量に要せず、装置・工程ごとのドメインずれを校正しやすい点に特徴がある。これは本件の拡張案であり、初期版の必須構成ではない。

(b) SEM 特徴量の prior 利用
SEM 画像そのものを必須入力にせず、面積、等価直径、円形度、長短比、エッジ/中心コントラスト等の低次元特徴量に圧縮して、AI 推論時の prior として用いる。JMONSEL では幾何ライブラリと測定 SEM 画像の整合による形状推定が行われ、SEM-EDS fusion の文献では SEM が EDS の空間情報補強に使われているため、SEM を EDX 推定の補助情報とする方向性自体は技術的に合理的である。もっとも、本件では SEM の役割を画像単独分類ではなく、探索空間縮小の補助に置く点が異なる。  ￼

(c) 高忠実度シミュレータによる補完
低信頼度サンプル又は重要サンプルのみ、DTSA-II や PENELOPE による高忠実度スペクトルを計算し、人工学習データの難例群、又は再判定用 basis として使う。DTSA-II は characteristic / continuum X-ray、self-absorption、EDS window absorption、peak broadening を含む EDS シミュレーションを提供し、PENELOPE は arbitrary materials を対象とする電子・光子輸送を提供する。  ￼

4-4. AI モデル仕様

基本方針
本件の発明の芯は「辞書主導の人工データ生成」と「アンカー整合性に基づく信頼度」であり、AI モデルは過度に複雑でなくてよい。したがって、好適例としては以下の Dictionary-Guided Multi-Label Network を想定する。

入力
	•	1D EDX スペクトル
	•	任意: SEM 低次元特徴量
	•	任意: 工程 ID / 装置 ID / beam energy

モデル構造（好適例）
	1.	多スケール 1D 畳み込み部
kernel size を複数持つ 1D CNN で、狭いピーク、広いピーク、重なりピークを同時に捉える。
	2.	アンカープーリング部
辞書に登録された各元素の energy_1~n の近傍窓を切り出し、元素ごとのアンカーベクトルを作る。
	3.	元素ヘッド
元素有無を multi-label で出力する。
	4.	成分比ヘッド
擬似成分比を回帰又は simplex 制約付き出力で出す。
	5.	信頼度ヘッド
アンカー一致数、出力確率のエントロピー、低 energy risk、近接ピーーク危険度を入力として信頼度を出す。

推奨する損失関数
\mathcal{L}
=
\lambda_1 \mathcal{L}_{\text{BCE}}
+
\lambda_2 \mathcal{L}_{\text{fraction}}
+
\lambda_3 \mathcal{L}_{\text{anchor-consistency}}
+
\lambda_4 \mathcal{L}_{\text{confidence-calib}}
	•	\mathcal{L}_{\text{BCE}}: 元素 presence の multi-label 損失
	•	\mathcal{L}_{\text{fraction}}: 擬似成分比の回帰損失
	•	\mathcal{L}_{\text{anchor-consistency}}: 同一元素内の複数アンカー整合性を促す損失
	•	\mathcal{L}_{\text{confidence-calib}}: 信頼度の校正損失

モデルの拡張候補
大量データが十分にある場合には、InceptionTime のような多スケール時系列 CNN や、混合スペクトル向けの CNN-Transformer ハイブリッドを参考に、注意機構を付与してもよい。InceptionTime は時系列分類で高いスケーラビリティを示し、ConInceDeep は重なりピークと弱いピークが 1D-CNN で十分に拾われにくいことを指摘し、FlavorFormer は複雑混合スペクトルで CNN と Transformer の併用有効性を示している。ただし、これらは EDX 直接の先行ではなく、混合スペクトル処理の設計参考としての位置付けである。  ￼

6. 提案手法による効果

6-1. 技術効果
	1.	少数実測でも初期モデルを立ち上げやすい
辞書だけで人工データを作れるため、ラベル付き実測が少ない段階でも学習開始できる。
	2.	副線欠落や重なりに強い
学習データ生成時に line dropout と overlap を意図的に増やすため、単純なピーク検出より実運用に近い失敗モードを事前学習できる。
	3.	説明可能な信頼度が出せる
「どの元素のどのアンカーが一致したか」を用いて信頼度を構成できるため、ブラックボックス性を下げられる。
	4.	再解析対象を絞れる
低信頼度のみを再評価に回すことで、DTSA-II や詳細 EDX の負荷を下げられる。

6-2. 効果の評価観点

効果	想定メカニズム	評価方法
同定率向上	line dropout/overlap 学習	micro/macro F1
主成分推定向上	multi-label + fraction 同時学習	top-1 accuracy, fraction MAE
見逃し低減	OR 判定 + アンカー整合	recall
工数削減	低信頼度のみ再評価	再評価率、1ロット工数
監査性向上	アンカー由来の説明出力	説明ログの監査容易性

6-3. 補足

既存公開例では、詳細 EDX 収集に 1〜2 分/欠陥を要する場合がある。したがって、本件のように高信頼度サンプルを自動確定し、低信頼度のみ詳細再評価する構成は、平均解析時間の削減に有効であると推定される。ただし、削減率は工程や欠陥分布に依存するため、量産ラインでの実測検証が必要である。  ￼

7-2. 進歩性の見立て（推定）

進歩性の芯は、単に「AI を使う」点ではなく、入力データの作り方と信頼度の作り方にあると整理するのがよい。
	•	入力データの作り方
元素ごとの複数ピーク候補を同一レコードに持つ簡易辞書を、人工データ生成と推論の両方で共通利用する点。
	•	特徴量/前処理
生スペクトル全体に加え、辞書アンカー近傍窓を別経路で扱う点。
	•	信頼度
モデル確率だけでなく、アンカー一致数、複数アンカー整合性、低 energy risk、基板混合疑いを統合する点。
	•	運用設計
低信頼度のみを詳細再評価へ回す段階構成を最初から含む点。

この整理であれば、既存の「SEM 画像分類」や「物理シミュレータ単体」や「ピーク fitting 単体」との違いを示しやすい。特に、公開特許では SEM 画像のみから composition-based classification を行う例や、成分分析でラベル付けした SEM 画像を学習する例が見られるが、本件の主戦場はEDX スペクトル学習であり、かつ人工データ生成規則そのものに発明の芯がある。  ￼

7-3. 進歩性の補強候補

進歩性を強めるには、以下のいずれかを明細書で具体化するとよい。
	1.	line dropout と overlap を意図的に増やす生成規則
	2.	同一元素内の複数アンカー整合損失
	3.	プロセス前後スペクトルによる校正
	4.	SEM を低次元特徴に圧縮して prior として使う
	5.	低信頼度のみ DTSA-II/PENELOPE に送るルーティング

8. 実施例（半導体製造での事例と従来の課題、その効果を説明）

8-1. 想定実施例

装置A はフッ素系プラズマ工程を含む半導体製造装置であり、ウェハ上 review で particle defect が検出される。対象粒子は、Y/O/F 系、Al/O/F 系、Si/O 系、Hf/Ta/W 系のいずれか、又はそれらの混合であると仮定する。これは、過去検討で想定した coating 起因・基板起因・装置材起因の代表元素群に対応する。なお、具体的な部材名・顧客情報・機密条件は匿名化し、本資料では装置A、工程B と表記する。

8-2. 従来の課題

従来は、各欠陥について EDX スペクトルを取得し、ピーク検出や fitting、必要に応じて追加計測を繰り返していた。このとき、Y 系や Hf/Ta/W 系の近接ピーーク、O/F の低エネルギー線、基板由来の Si/O が混在し、解析が安定しにくい。また、既存公開例でも EDX 収集に 1〜2 分/欠陥を要する場合があるため、欠陥数が多いと工数が問題になる。  ￼

8-3. 本技術の適用
	1.	装置A向け辞書として、O,F,Al,Si,Y,Hf,Ta,W を含む簡易辞書を作成する。
	2.	当該辞書から、line dropout、peak overlap、energy drift、background variation、substrate mixture を加えた人工 EDX スペクトルを大量生成する。
	3.	人工データで AI モデルを学習する。
	4.	実測 EDX を入力し、候補元素、擬似成分比、信頼度を出力する。
	5.	低信頼度のみ詳細再評価に回す。必要に応じて clean wafer 参照又は DTSA-II basis で再評価する。DTSA-II は Filter-fitting と XPP を備え、実測比較可能な analytical/Monte Carlo スペクトルを生成できる。  ￼

8-4. 効果（試算・要検証）

仮に、対象欠陥の 60% を高信頼度で一次判定でき、残り 40% のみを詳細再評価へ回すとすると、詳細 EDX 収集・再解析の対象数は 40% に圧縮できる。既存公開例の 1〜2 分/欠陥という条件を基準にすると、平均工数は概算で 30〜60% 程度低減し得る。これは試算であり、実際の効果は beam 条件、欠陥サイズ、基板混合の程度、装置Aの測定フローに依存する。  ￼

10. 本技術により創出される新たな価値

10-1. 従来困難だった意思決定（何が初めて可能になるか）
	•	全欠陥を同一深さで解析するのではなく、信頼度に応じて層別処理する判断
	•	ラベル付き実測が少ない段階でも、装置A向け初期モデルを立ち上げる判断
	•	SEM を画像単独判定に使うのではなく、EDX の探索空間を絞る prior として使う判断
	•	プロセス前後スペクトルを校正資産として再利用する判断

10-2. 半導体製造装置での EDX 計測に対するシミュレーションによる人工データ生成の価値

人工データ生成の価値は、単にデータ数を増やすことではない。実際に困る失敗モードを意図的に増やせることが重要である。具体的には、近接ピーーク、弱い副線、低エネルギー吸収、基板混合、エネルギードリフトを制御的に入れることで、モデルが「簡単なスペクトル」だけでなく「現場で問題になるスペクトル」を学習できる。physics-guided NMF や espm が示すように、物理を含む人工データは混合・低 S/N 条件で有効性を持ち得る。  ￼

10-3. SEM, EDX 実測データを組み込む価値

SEM と EDX を組み合わせる価値は、SEM を EDS の代替にすることではなく、SEM で幾何・形態の探索空間を狭め、EDX で成分を絞る役割分担にある。JMONSEL では geometry library と測定画像の整合で形状パラメータを推定し、SEM-EDS fusion では低解像度 EDS に対し SEM が補助情報になることが示されている。したがって、本件でも SEM を prior として使う拡張は合理的である。  ￼
