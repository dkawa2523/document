1. 1ページ要約（発明の要旨／新規性の核3点／期待効果の定量）

本技術は、低圧プラズマのサロゲート学習において、境界の位置を表す役割別 SDF と、境界条件の内容を表す境界物理マップとを、単一の U-Net に一体入力する構成である。通常の U-Net は局所化と広域文脈取得に強いが、低圧プラズマでは壁近傍の RF sheath、誘電体界面、ガス導入口、排気口が解を強く支配するため、単なるマスクや少数スカラーでは境界条件を十分に表現しにくい。低圧プラズマでは RF sheath が材料表面近傍で形成され、ガス流やポンプ条件も内部状態を変えることが知られている。  ￼

本発明の中心は、次の 3 点である。
	1.	役割別 multi-SDF の固定チャネル化
接地境界、印加電圧境界、浮遊境界、誘電体、ガス入力、排気といった境界の役割ごとに SDF を生成し、境界数や形状が変わっても入力仕様を固定する。
	2.	境界条件の空間マップ化
入力電圧は \Phi_{\mathrm{bc}} などの電圧影響場、誘電率は \epsilon_r マップ、ガス入力は inflow map、排気は suction map として空間分布に変換し、U-Net に「どこにあるか」と「何をするか」を同時に与える。
	3.	低圧プラズマ向けの境界特化学習
壁帯域、誘電体界面帯域、入口帯域、出口帯域を SDF で切り分け、壁法線電場、誘電体 jump、port balance を用いることで、通常 U-Net が苦手な境界近傍精度を改善する。

新規性の核 3 点

核	内容	従来との差
核1	役割別 multi-SDF を固定チャネルで入力	パーツ ID 依存や boundary 数依存を下げる
核2	電圧・誘電率・流入・排気を境界物理マップとして空間化	scalar 条件をそのまま入れる方式と異なり、空間作用を直接表現
核3	壁・界面・port 向けの帯域学習と後処理拘束	通常 U-Net の全域平均誤差最小化から脱却

期待効果の定量（推定／要実証）

以下は設計目標レンジであり、実験値ではない。

指標	通常 U-Net 比	期待効果
壁近傍 band の電位/密度 RMSE	20〜50% 改善	sheath 位置と勾配の再現性向上
誘電体界面近傍の電場誤差	25〜60% 改善	\epsilon_r 差による界面電束変化を保持
inlet/outlet 条件の未学習組合せでの外挿誤差	15〜40% 改善	圧力だけでなく流量・排気能力差を区別
形状変更時の再学習負荷	20〜40% 低減	役割別 SDF により入力仕様を再利用
フルシミュレーション代替の設計探索速度	10〜100 倍の候補絞り込み	推定：用途依存、要実証

2. 従来技術（背景）と先行技術カテゴリ（従来技術は通常のUnetベースを想定）

通常の U-Net は、contracting path と symmetric expanding path により、広域文脈と局所位置を両立する畳み込みエンコーダ・デコーダである。近年の PDE 向け modern U-Net では、Residual block、GroupNorm、GELU、scale-and-shift conditioning などが使われている。  ￼

また、幾何が変化する PDE surrogate では、SDF により計算領域形状を符号付き距離として表現する手法が提案されている。さらに、座標を明示チャネルとして渡す CoordConv、条件スカラーを特徴へ affine 注入する FiLM、係数不連続や可変境界条件を入力特徴として扱う interface/operator learning も知られている。  ￼

本資料では、比較対象となる従来技術を「通常 U-Net ベース」として、以下のように整理する。

先行技術カテゴリ	典型構成	低圧プラズマへの適用時の弱点
C1: 通常 U-Net 回帰	幾何マスク + 材料マスク + プロセススカラー	境界条件の空間作用を十分に表せない
C2: SDF 形状条件付け	SDF を入力し形状変化に対応	位置表現は強いが、電圧・流入・排気などの内容は別途必要
C3: CoordConv/FiLM 型条件付け	座標チャネル、条件埋め込み	境界条件を空間物理として表す仕組みが必要
C4: Interface/operator 学習	係数・境界条件を入力特徴化	高機能だが構造が重くなりやすい

本発明の立ち位置は、C1 の通常 U-Net をベースラインにしつつ、C2〜C4 の有効要素を低圧プラズマ向けに最小構成で統合する点にある。

3. 従来の課題（発生条件、現行対策の限界、評価指標、制約）

低圧プラズマでは、壁近傍に sheath が形成され、RF 駆動時には RF sheath が表面近傍の薄い境界層として振る舞う。表面反射が十分強いと Debye sheath 自体が大きく変化し得るし、表面帯電や SEE も壁電位や sheath ポテンシャルを変え得る。そのため、壁近傍は bulk と同じ統計・同じ表現で学習すると精度が落ちやすい。  ￼

誘電体が存在する場合、問題は単なる金属壁ではなく係数不連続を持つ界面問題になる。したがって、通常 U-Net のように材料 one-hot と全域 loss だけで扱うと、\epsilon_r の違いで生じる界面近傍の電場屈曲や jump 条件が平滑化されやすい。界面問題の機械学習では、係数、ソース、境界条件を入力特徴として扱う設計が有効とされている。  ￼

ガス導入と排気も同様に、単なる chamber pressure 1 個では不十分である。低圧 ICP では gas flow rate の変化により electron density が増加し electron temperature が低下することが報告されており、また outlet 近傍で測った pressure は bulk pressure を過小評価し得る。加えて、真空系では effective pumping speed は conductance により制約され、pump の公称値だけでは chamber から見た排気能力を表し切れない。  ￼

現行対策としては、マスクの多チャネル化、材料 one-hot、全域 MSE 重み付け、境界近傍 crop 強化などがある。しかし、これらは境界がどこにあるかには効いても、その境界が何をするかまでを空間的に表現しないことが多い。そのため、電圧変更、誘電率変更、流量変更、排気構成変更が入ると、通常 U-Net は条件間の作用差を内部で再構成しなければならず、境界近傍で誤差が残りやすい。

本発明が重視する評価指標は、全域 RMSE だけではなく、次のような境界別評価である。

評価項目	内容
壁帯域誤差	壁から一定距離内の φ, ne, ni, Te 誤差
法線電場誤差	E_n = \mathbf n \cdot \nabla \phi の誤差
誘電体界面残差	[\phi], [\epsilon \partial_n \phi] の残差
inlet/outlet 整合	総 inflow / outflow、port 近傍分布の整合
条件外挿誤差	未学習の電圧・\epsilon_r・流量・排気条件への追従

制約として、半導体装置では教師データ生成が高コストであり、低圧では Knudsen 数の影響で kinetic/hybrid モデルが必要になる場合もある。このため、複雑すぎる multi-model 構成ではなく、単一 U-Net に境界表現を強く載せる方が実装・運用上有利である。  ￼

4. 提案手法

4-1. システム構成（文章）

本発明のシステムは、主として以下の機能部から構成される。
	1.	境界役割分類部
CAD、メッシュ、装置定義から、境界を「接地」「印加電圧」「浮遊」「誘電体」「ガス入力」「排気」の役割に分類する。
	2.	SDF 生成部
前記役割ごとに SDF を生成する。境界数や形状が変化しても、役割数が同じであれば入力チャネル数を固定できる。
	3.	境界物理マップ生成部
入力電圧、誘電率、ガス入力条件、排気条件を、それぞれ空間マップへ拡張する。
例として、電圧は \Phi_{\mathrm{bc}}、誘電率は \epsilon_r map、ガス入力は inflow map、排気は suction map とする。
	4.	特徴テンソル生成部
座標、SDF、法線、境界物理マップを一つの固定仕様テンソルへ統合する。
	5.	SDF 条件付き U-Net
前記特徴テンソルを入力として、プラズマ状態場を直接推定する。
任意で、スカラー条件を条件埋め込みして各 residual block に scale-and-shift で注入してもよい。
	6.	後処理部
既知の電圧境界について、推論後に境界値を上書きする。
任意で、信頼度推定部を備え、低信頼時に教師シミュレーション再実行や再学習へ分岐してもよい。

4-2. 特徴量化仕様（SDFでの内容）

4-2-1. 役割別 SDF
本発明の基本入力は、役割別 SDF である。

d_{\mathrm{gnd}},\;
d_{\mathrm{pow}},\;
d_{\mathrm{float}},\;
d_{\mathrm{diel}},\;
d_{\mathrm{in}},\;
d_{\mathrm{out}}

ここで各 d_r(\mathbf x) は、座標 \mathbf x における、役割 r の最近傍境界までの符号付き距離である。

4-2-2. 法線と帯域
最も近い境界役割を

r^\* = \arg\min_r |d_r(\mathbf x)|

とし、その法線を

\mathbf n(\mathbf x)=\frac{\nabla d_{r^\*}(\mathbf x)}{\|\nabla d_{r^\*}(\mathbf x)\|}

で与える。
また、壁帯域・界面帯域・入口帯域・出口帯域は、各 SDF に閾値 \tau_r を設けることで作る。

4-2-3. 境界物理マップ
本発明では、SDF に加え、境界条件を以下のように空間マップへ変換する。

区分	記号例	内容	備考
電圧影響場	\Phi_{\mathrm{bc}}	印加電圧が内部へ与える基底場	必須
電場影響場	\mathbf E_{\mathrm{bc}}	-\nabla\Phi_{\mathrm{bc}}	推奨
誘電率	\epsilon_r	パーツごとの誘電率分布	必須
流入強度	Q_{\mathrm{in}}	mass-flow または pressure inlet の空間化	必須
流入組成	Y_{\mathrm{in},s}	主要ガス種ごとの空間化	任意
流入温度	T_{\mathrm{in}}	中性ガス温度	任意
排気強度	S_{\mathrm{out}}	有効排気能力の空間化	必須
壁物性	P_{\mathrm{wall}}	SEE、反射率、sticking 等	任意

4-2-4. 電圧影響場の生成
印加電圧条件は scalar のまま与えず、次式の解として空間化する。

\nabla \cdot (\epsilon_r(\mathbf x)\nabla \Phi_{\mathrm{bc}}(\mathbf x,t)) = 0

境界条件は、接地境界、印加電圧境界、絶縁境界等に応じて設定する。
これにより、電極数や配置が変わっても、U-Net には 1 枚または少数枚の固定チャネルとして与えられる。

4-2-5. ガス入力・排気の空間化
ガス入力と排気は、port 境界値を内部へ拡張したマップで表す。拡張関数は、最近傍境界拡張、調和拡張、または他の決定的拡張でよい。

Q_{\mathrm{in}}(\mathbf x)=\mathrm{Ext}_{\Gamma_{\mathrm{in}}}(\dot m_{\mathrm{in}} \text{ または } p_{\mathrm{in}})

S_{\mathrm{out}}(\mathbf x)=\mathrm{Ext}_{\Gamma_{\mathrm{out}}}(S_{\mathrm eff})

ここで

S_{\mathrm eff}=\frac{SC}{S+C}

は pump speed S と conductance C から得られる有効排気能力である。真空系では pumping speed は aperture conductance を超えられず、effective pumping speed は conductance に強く制約される。  ￼

4-3. 教師データ/ラベル定義

仮定として、教師データはシミュレーションスナップショットであり、各サンプルは以下を含む。

区分	内容
幾何入力	CAD/メッシュ、役割分類情報
条件入力	電圧、\epsilon_r、ガス入力、排気条件
入力特徴	SDF 群、法線、境界物理マップ、座標
主ラベル	φ, ne, ni, Te
補助ラベル	E_n、界面 jump、port throughput、必要なら電流・流束

教師シミュレーションは、要確認として、流体、PIC、ハイブリッドのいずれでもよい。低圧では kinetic/hybrid の方が正確な場合があるが計算負荷が高いため、本発明は教師ソルバ種別を限定しない。  ￼

データ分割は、ランダム画素単位ではなく、幾何条件・境界条件組合せ単位で行うことが望ましい。これにより、真の一般化性能、すなわち「未学習条件への追従」を評価できる。

4-4. AIモデル仕様

4-4-1. ネットワーク
本発明の推奨構成は、4 段程度の residual U-Net である。
	•	Encoder-Decoder 型
	•	Skip connection あり
	•	各段に 2 個程度の residual block
	•	GroupNorm
	•	GELU
	•	3×3 convolution
	•	入力座標チャネルあり
	•	任意で scalar condition encoder + scale-and-shift conditioning

座標チャネルは、畳み込みが位置依存タスクで失敗し得ることを補う目的で導入する。条件スカラーの feature-wise affine 注入は FiLM 系に相当し、modern U-Net の条件付けとも整合する。  ￼

4-4-2. 出力
本発明では、出力は差分ではなく直接物理量とする。

\hat y = (\hat\phi,\hat n_e,\hat n_i,\hat T_e)

既知の電圧境界についてのみ、推論後に境界値上書きを実施する。
これにより、学習対象は「境界値そのもの」ではなく「境界から内部への接続関係」になる。

4-4-3. 損失関数
最小構成として、次式を採用する。

L = \lambda_f L_{\mathrm{field}}
+ \lambda_w L_{\mathrm{wall}}
+ \lambda_d L_{\mathrm{diel}}
+ \lambda_p L_{\mathrm{port}}
	•	L_{\mathrm{field}}: 全域の φ, ne, ni, Te 誤差
	•	L_{\mathrm{wall}}: 壁帯域での法線電場、必要なら壁近傍場誤差
	•	L_{\mathrm{diel}}: 誘電体界面における continuity / jump 残差
	•	L_{\mathrm{port}}: inflow / outflow 整合、port 近傍分布整合

誘電体界面については、少なくとも

[\phi]=0,\qquad [\epsilon \partial_n \phi]=0

を用いる。
任意拡張として、表面帯電を状態変数 \sigma_s で持つ場合には

[\epsilon \partial_n \phi]=\sigma_s

としてもよい。

4-4-4. 学習方法
	•	SDF 帯域による層別サンプリング
	•	wall / dielectric / inlet / outlet / bulk の各帯域を毎バッチに含める
	•	評価は全域 RMSE ではなく帯域別に実施
	•	任意で、低信頼サンプルを再学習集合へ戻す active learning を行う

6. 提案手法による効果（通常Unetに対するSDF,物性ごとの効果をそれぞれ説明）

6-1. 効果の総括

通常 U-Net は、入力テンソル上の局所パターンから解を近似するため、境界条件が「空間にどう作用するか」が明示されないと、境界近傍の sharp transition を平均化しやすい。本発明は、SDF による位置表現と境界物理マップによる作用表現とを同時に入れることで、この弱点を補う。

6-2. 項目別効果

要素	通常 U-Net	本発明	効果
役割別 SDF	なし、または単純マスク	接地・印加・浮遊・誘電体・入口・出口を SDF 化	境界距離と方向が明示され、壁近傍の位置誤差を低減
法線情報	暗黙的	SDF 勾配から生成	sheath の主方向である法線方向勾配を捉えやすい
入力電圧	scalar 条件	\Phi_{\mathrm{bc}}, \mathbf E_{\mathrm{bc}}	電極配置・形状・誘電率越し結合を空間的に反映
誘電率	one-hot 程度	\epsilon_r map + 界面 loss	界面電束変化を保持
ガス入力	chamber pressure や総流量	inflow map	inlet 位置、向き、組成差を分離
排気	outlet pressure	suction map + S_{\mathrm eff}	排気構成差、duct 制約差を表現
既知電位拘束	soft に学習	推論後上書き	既知境界の誤差を除去
学習単位	全域平均	SDF 帯域学習	壁・界面・port 精度を重点改善

6-3. SDF の効果

SDF を入れるだけでも、境界までの距離と法線方向が明示されるため、単純マスクより境界位置の再現が安定する。形状条件付き surrogate では、SDF による幾何表現が有効であることが報告されている。  ￼

6-4. 入力電圧の効果

電圧を scalar ではなく \Phi_{\mathrm{bc}} に変換すると、同じ 100 V でも「どの電極から」「どの誘電体を介して」「どこへ届くか」が明示される。RF sheath は材料表面近傍で形成されるため、この空間表現は通常 U-Net より壁近傍精度に効きやすい。  ￼

6-5. 誘電率の効果

\epsilon_r map と界面 loss を加えることで、誘電体窓、絶縁リング、スペーサ等の近傍での電場曲がりや jump 残差を抑えやすい。界面問題では、係数・境界条件を入力特徴として扱うことが有効とされている。  ￼

6-6. ガス入力の効果

低圧 ICP では gas flow rate が electron density と electron temperature を変えるため、pressure 1 個より inflow map の方が状態差を分離しやすい。  ￼

6-7. 排気の効果

排気は outlet pressure のみではなく、pump speed と conductance の組合せで実効性能が決まる。したがって suction map と \(S_{\mathrm eff}}\) を与えると、排気構成差を U-Net がより直接学べる。 outlet 近傍圧力は bulk pressure を過小評価し得るため、この点でも有利である。  ￼

7. 新規性・進歩性

7-1. 新規性の候補点

候補点 A
境界をパーツ ID ではなく役割別 multi-SDFとして固定チャネル化する点。
これにより、境界数が変わっても U-Net の入力仕様を維持できる。

候補点 B
入力電圧、誘電率、ガス入力、排気を、それぞれ境界物理マップに変換して SDF と併用する点。
つまり、「境界の場所」と「境界の作用」を分離かつ統合して表現する。

候補点 C
低圧プラズマに特有な壁帯域・界面帯域・port 帯域に対する学習と、既知電位上書き、界面 jump、port balance を組み合わせる点。

7-2. 進歩性の論点

個別要素としては、U-Net、SDF 形状条件付け、CoordConv/FiLM、interface operator learning はそれぞれ既知カテゴリに属する。  ￼

しかし、本発明はそれらを単に寄せ集めるのではなく、半導体製造向け低圧プラズマという特定用途に対して、
	•	境界役割を固定 schema に整理し、
	•	可変境界条件を空間マップ化し、
	•	単一の U-Net で扱い、
	•	壁・界面・port という物理的に異質な帯域を別々に最適化する、

という構成を採る。
この構成は、通常 U-Net の「境界条件をスカラーまたは単純マスクで与える」設計とは発想が異なる。

7-3. 本調査時点での評価

限定的な技術調査の範囲では、個別要素は確認できた一方、
「役割別 multi-SDF + 電圧影響場 + 誘電率界面 + inflow/outflow map + U-Net + 境界帯域学習」
という組合せを、半導体低圧プラズマサロゲート向けに直接開示する資料は未確認である。

ただし、これは正式な先行技術調査の代替ではなく、要本格サーチである。

7-4. 特許上の芯として推奨する表現

最も強い芯は、U-Net そのものではなく、
「境界役割別 SDF と境界物理マップを併用した固定仕様入力によって、可変な装置境界条件を単一 surrogate に吸収する」
点に置くのが望ましい。

8. 実施例（最低2つ：代表ケース＋変形例。条件・手順・結果が書ける範囲で）

実施例1：装置Aにおける代表ケース

前提
仮定：装置Aは、1つの印加電圧電極、接地チャンバー壁、誘電体窓、1系統のガス導入部、1系統の排気部を有する低圧プラズマ処理装置である。

条件
	•	出力対象：φ, ne, ni, Te
	•	教師：シミュレーションスナップショット
	•	可変条件：電極電圧、誘電率、流量、ガス組成、排気能力

手順
	1.	境界役割を接地、印加、誘電体、入口、出口に分類する。
	2.	各役割の SDF を生成する。
	3.	\epsilon_r map を生成する。
	4.	境界電圧から \Phi_{\mathrm{bc}} と \mathbf E_{\mathrm{bc}} を生成する。
	5.	流量・組成・温度から inflow map を生成する。
	6.	pump speed と conductance から S_{\mathrm eff} を算出し suction map を生成する。
	7.	前記特徴テンソルを residual U-Net に入力し、φ, ne, ni, Te を推定する。
	8.	既知電位境界を上書きする。
	9.	壁帯域、界面帯域、port 帯域で評価する。

結果（推定）
通常 U-Net に対し、少なくとも壁帯域と誘電体界面帯域で誤差低減が見込まれる。特に、電圧 sweep 時の sheath 形状変化、誘電体窓近傍の電場分布、入口/出口条件変更時の非対称場に対して安定性向上が期待される。

実施例2：装置Bにおける変形例

前提
仮定：装置Bは、複数の印加電圧境界、複数のガス導入口、複数の排気開口を有し、境界数がロットまたは部材交換で変動する。

条件
	•	境界数は可変
	•	役割数は固定
	•	電圧や流量は境界ごとに異なる

手順
	1.	複数の印加電圧境界を個別に扱うのではなく、境界条件に応じた \Phi_{\mathrm{bc}} へ集約する。
	2.	複数 inlet の条件は、合成 inflow map 群に変換する。
	3.	複数 outlet の条件は、合成 suction map に変換する。
	4.	SDF チャネル数は役割数に応じて固定とする。
	5.	代表ケースと同様に U-Net を用いて推定する。

結果（推定）
境界数が変わっても、U-Net 本体の入出力仕様を変更せずに適用できる。
このため、装置改造、部材追加、排気経路変更を伴う設計探索で、再設計コストを抑制できる。

10. 本技術により創出される新たな価値、活用事例

10-1. 従来困難だった意思決定（何が初めて可能になるか）
	1.	境界条件変更を含む高速設計探索

	2.	部材変更影響の即時比較

	3.	装置差をまたぐ recipe 検討
